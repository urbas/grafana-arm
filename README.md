# Grafana ARM ![dockerhub image badge]
Image containing Grafana compiled for ARM.

Get it from Docker hub: https://hub.docker.com/r/urbas/grafana/tags/

## Usage
```
sudo docker run -p 80:3000 urbas/grafana
```
